# Nodejs API Tasks

## Commands

### How to run

```
npm install 
```

```
npm start 
```

To test it, there is a Postman collention on collection folder to test it if you want. Otherwise, go to localhost:3000.

### How to test

```
npm test 
```

To check the coverage
```
npm run coverage 
```
